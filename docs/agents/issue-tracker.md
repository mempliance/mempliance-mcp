# Issue Tracker

Issues for this project are tracked on GitHub at:
https://github.com/mempliance/mempliance-mcp/issues

## Creating issues

Use the GitHub CLI or web UI to create issues. Include:
- A clear title
- Steps to reproduce (for bugs)
- Expected vs actual behaviour
- Relevant labels from the triage vocabulary

## Querying issues

```bash
gh issue list --repo mempliance/mempliance-mcp
gh issue view <number> --repo mempliance/mempliance-mcp
```

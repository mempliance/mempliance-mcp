# Triage Labels

Default five-label vocabulary used for issues at
https://github.com/mempliance/mempliance-mcp.

| Label | Meaning |
|-------|---------|
| `bug` | Something is not working as documented |
| `enhancement` | New feature or improvement to existing behaviour |
| `question` | Clarification needed; no code change expected |
| `documentation` | Docs-only change (README, docstrings, guides) |
| `blocked` | Cannot proceed; waiting on external dependency or decision |

Apply exactly one primary label per issue. A second label may be added when the
primary alone is ambiguous (e.g. `bug` + `documentation` for a docs error).
